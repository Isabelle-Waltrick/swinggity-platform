# swinggity-platform

Web platform for Swinggity.

## Live
- https://swinggity-platform.vercel.app

## Tech stack
- JavaScript
- CSS

## Getting started

### Prerequisites
- Node.js (LTS recommended)
- npm (comes with Node)

### Install
```bash
npm install
```

### Run locally
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```

## Project structure
- `client/` — frontend app
